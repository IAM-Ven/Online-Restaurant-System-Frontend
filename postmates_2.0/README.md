## To run locally

### Clone repository
`git clone https://github.com/syedsadman16/Online-Restaurant-System-Frontend.git`

### Move into project directory
`cd postmates_2.0`

### Install packages (creates node_modules folder)
`npm install`

### Startup localhost and run
`npm start`
